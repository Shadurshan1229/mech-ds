Usage Rules

Sharp Geometry
  • radius:0 on all elements by default
  • Toggle track: only round exception
  • Rounded buttons or cards
  • radius-md, radius-lg anywhere
Flat Surface
  • Depth via border + bg contrast only
  • Hover: border or fill change
  • Active: 2px orange border
  • box-shadow on cards or buttons
  • drop-shadow or filter
The Orange Rule
  • One CTA per view maximum
  • Active borders and indicators
  • Progress fill and data signals
  • Large background fills
  • Decoration of any kind
Button Font
  • All buttons: Space Grotesk always
  • Link buttons: in-context secondary actions
  • Poppins on any button
  • Link buttons as primary CTAs
Monospace
  • All counts, IDs, percentages
  • Timestamps, durations, versions
  • Body text or headings
Uppercase
  • Status tags, section headers
  • Breadcrumbs, table headers
  • Form labels
  • Page titles, button labels, errors
